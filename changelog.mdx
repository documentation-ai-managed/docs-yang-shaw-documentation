## How to Stay Updated

Follow these steps to keep your integration current:

<Steps>
  <Step title="Subscribe to Updates" icon="bell">
    Visit the notifications page and enable changelog alerts.
  </Step>
  <Step title="Check Version" icon="package">
    Query your API response for the current version:
    
    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const response = await fetch('https://api.cheneycommunications.com/v1/status');
      const version = response.version; // e.g., "0.3.0"
      console.log(`Current version: ${version}`);
      ```
      ```python
      import requests
      response = requests.get('https://api.cheneycommunications.com/v1/status')
      version = response.json()['version']  # e.g., "0.3.0"
      print(f"Current version: {version}")
      ```
    </CodeGroup>
  </Step>
  <Step title="Test Changes" icon="play">
    Use our sandbox environment to validate updates before production.
  </Step>
</Steps>
